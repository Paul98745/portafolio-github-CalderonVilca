# Mi primer portafolio versionado en GitHub
Aplicación web de lista de tareas (to-do list) con HTML, CSS y JavaScript puro. Permite agregar, completar, eliminar y filtrar tareas, con persistencia usando localStorage.

# Tecnologías:
HTML5, CSS3, JavaScript 
Google Fonts (Syne + DM Mono)
Git & GitHub
